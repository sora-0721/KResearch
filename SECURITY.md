# Security Policy | 安全策略

## Supported Versions | 支持的版本

We currently only provide security updates for the latest main branch (`main`).

目前我们仅对项目的最新主分支 (`main`) 提供安全更新支持。

| Version | Supported |
| :--- | :--- |
| `main` (latest) | ✅ Supported |
| Historical versions | ❌ Not Supported |

| 版本 | 支持状态 |
| :--- | :--- |
| `main` (最新版) | ✅ 支持 |
| 历史版本 | ❌ 不支持 |
